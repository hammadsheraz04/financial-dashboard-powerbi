# Financial Performance Dashboard — Power BI

P&L performance dashboard covering 24 months of revenue data 
across 4 Pakistani cities (Karachi, Lahore, Islamabad, Peshawar).

## What's inside
- Star schema data model (Actuals + Budget + Dim_Date + Dim_Region)
- 10+ DAX measures: Revenue YTD, GP%, YoY Growth, Budget Variance
- Executive overview page: KPI cards, trend line, regional breakdown
- Variance analysis page: red/green heatmap matrix, waterfall chart
- Interactive slicers: Year and Region filters synced across pages

## Key findings
- Revenue grew 6% YoY (2024 vs 2025)
- All regions missed budget by 4.76% consistently
- Karachi generated highest total revenue across both years
- Budget vs Actual gap visible in monthly trend line

## Tools used
Power BI Desktop · Excel · DAX · Power Query · Star Schema Modelling
